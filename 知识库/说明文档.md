# 产品定位

一款AI笔记产品~ 。

不会追求成为通用的、面向所有职业角色的产品，产品的设计思路是让最新的AI技术帮助用户更好的输入+输出，笔者坚信最好的学习路径就是高质量的输入+高频次的输出，所以本产品适合所有想借助AI更好的"充电"的用户~

当然如果你是自媒体创作者、程序员，亦或者你使用过cursor等AI编程产品，使用本产品会更加丝滑。

产品未来更新会持续追求：更强大、更懂你的AI辅助、更多实用的围绕高质量『输入+输出』的功能，更纯净的交互界面。
